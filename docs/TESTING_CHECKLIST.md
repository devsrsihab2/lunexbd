# Testing Checklist

- Responsive UI: desktop, tablet, mobile, header, footer, product grid, product detail, cart, checkout, account pages.
- Product: listing, detail, category filtering, search, sorting, pagination, variations, stock.
- Cart: add, update quantity, remove, empty state, totals, coupon apply/remove.
- Checkout: guest enabled/disabled, logged-in checkout, billing/shipping validation, shipping method, tax, coupon, COD, payment redirect, confirmation.
- Auth: login, register, logout, protected routes, session expiry, invalid credentials.
- Account: dashboard, order history, order detail, timeline, IDOR protection.
- Wishlist: logged-in sync, guest enabled/disabled, add/remove, move to cart.
- CMS: homepage, menus, footer, blog list/detail, static pages.
- SEO: titles, descriptions, canonicals, OG tags, schema, sitemap, robots.
- Performance: image optimization, lazy loading, API caching, bundle size, Lighthouse.
- Security: input validation, XSS sanitization, CSRF if cookie auth, rate limiting, credential safety.
